# CUG_Campus-navigation
### 体验可前往微信小程序搜索：地大校园导航
### 中国地质大学（武汉）南望山、未来城校区微信小程序校园导航，使用的是腾讯地图接口和插件，所以不具备技术性，纯属娱乐，不喜勿喷。
